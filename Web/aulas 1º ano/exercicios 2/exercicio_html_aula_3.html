1 Exercício 1 (comentários)

Remova as tags de comentário do elemento p. 

<!DOCTYPE html>
<html>
<body>

<!-- <p>This is a paragraph.</p> -->

</body>
</html>

2 Exercício 2 (comentários)

Comente o texto "Não mostrar esse texto no navegador" 

<!DOCTYPE html>
<html>
<body>

<p>Esse é um parágrafo.</p>
Não mostrar esse texto no navegador
<p>Esse é outro parágrafo.</p>

</body>
</html>

3 Exercício 1 (links)

Faça com que o link abaixo aponte para https://www.w3schools.com: 

<!DOCTYPE html>
<html>
<body>

<a>Visite nosso tutorial HTML.</a>

</body>
</html>

4 Exercício 2 (links)

Transforme o link abaixo em um link local: 

<!DOCTYPE html>
<html>
<body>

<a href="https://www.w3schools.com/html/html_images.asp">Imagens em HTML</a>

</body>
</html>

5 Exercício 3 (links)

Faça com que o link abaixo abra em uma nova janela: 

<!DOCTYPE html>
<html>
<body>

<a href="html_images.asp">HTML Images</a>

</body>
</html>

6 Exercício 4 (links)

Adicione um link à imagem abaixo, faça com que o usuário vá para http:www.google.com ao clicar no link: 

<!DOCTYPE html>
<html>
<body>

<img src="https://www.w3schools.com/html/smiley.gif" alt="HTML tutorial" width="42" height="42">

</body>
</html>
