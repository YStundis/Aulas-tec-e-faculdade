1 Exercício

Adicione a tag <style> dentro do elemento <head> no documento abaixo, com a seguinte regra: 

p { color: red; }

<!DOCTYPE html>
<html>
<body>

<h2>The style Attribute</h2>
<p>The style attribute is used to specify the styling of an element, like color:</p>

<p>I am a paragraph.</p>

</body>
</html>

2 Exercício

Adicione a regra abaixo na tag <style>: 

h1 { font-size:60px; }

Na página abaixo: 

<!DOCTYPE html>
<html>
<head>
   <style>
   </style>
</head>
<body>

<h1>Alterando o tamanho da fonte</h1>

<p>É possível alterar o tamanho de um título com CSS usando a propriedade font-size.</p>

</body>
</html>

3 Exercício

Insira as seguintes regras CSS na tag <style>: 

h1 { font-family:verdana; }
p { font-family:courier; }

<!DOCTYPE html>
<html>
<head>
<style>

</style>
</head>
<body>

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

</body>
</html>
