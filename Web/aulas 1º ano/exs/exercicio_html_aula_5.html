1 Exercício 8 (css)

Altere a cor do pano de fundo da página para amarelo (yellow): 

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color:lightgrey}
h1   {color:blue}
p    {color:black}
</style>
</head>
<body>

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>

</body>
</html>

3 Exercício 10

<!DOCTYPE html>
<html>
<head>
<style>

</style>
</head>
<body>

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>
<p>This is also paragraph.</p>
<p>This is a third paragraph.</p>

</body>
</html>

5 Exercício 12

Adicione um id com o valor "especial" ao último parágrafo: 

<!DOCTYPE html>
<html>
<head>
<style>
p#especial {
    background-color:black;
    color:white;
}
</style>
</head>
<body>

<h1>This is a heading</h1>

<p>This is a paragraph.</p>
<p>This is also paragraph.</p>
<p>I want to be a special paragraph!</p>

</body>
</html>

6 Exercício 13

Adicione a class "intro" a todos os elementos <p> 

<!DOCTYPE html>
<html>
<head>
<style>
p.intro {
    background-color:black;
    color:white;
    border:1px solid grey;
    padding:10px;
    margin:30px;
    font-size:150%;
}
</style>
</head>
<body>

<h1>This is a heading</h1>

<p>This is a paragraph.</p>
<p>This is a paragraph.</p>
<p>This is a paragraph.</p>

</body>
</html>
